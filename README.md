# AI Daily News HTML 技能介绍

## 📖 开发背景

这个技能起源于一个真实的用户痛点：缺乏页面版的每日新闻日报推送功能。在日常工作和学习中，用户需要快速了解每日最新的 AI 领域动态，但现有的新闻推送多为纯文本格式，缺乏美观、易读的页面展示方式。

本技能结合了两个现有技能的优势，为用户提供一站式的解决方案：既能获取精准的 AI 新闻内容，又能生成美观易读的 HTML 页面，方便用户阅读和分享。

---

## 🎯 主要功能

### 1. 智能新闻搜索
- **核心关注领域**：
  - AI 领域：大模型版本更新、AI Agent 形态、多模态技术突破、AI Native 应用创新
  - 音视频产品：AI 视频生成、实时流交互、平台产品改版
  - 边缘计算：端侧大模型、本地推理算力突破、IoT 落地场景
  - 核心大厂动态：OpenAI、Google、Apple、Meta、字节跳动、阿里巴巴等头部公司的产品发布与战略

### 2. 结构化新闻分类
将新闻归类到 6 个专业分类：
- 模型与产品发布
- Agent 与应用落地
- 企业与商业化动态
- 基础设施 / 芯片 / 算力
- 政策 / 监管 / AI 安全
- 研究进展与技术趋势

### 3. AI 智能摘要
生成三种风格的新闻摘要：
- **故事性风格**：用生动的语言串联当天新闻
- **简洁版风格**：分点列出核心要点
- **专业版风格**：正式专业的深度解读

### 4. 热点排行榜
基于热度评分自动生成今日热点 Top 6 排行榜

### 5. 美观的 HTML 页面
- 响应式设计，适配电脑端和手机端
- 清晰的侧边栏导航
- 丰富的视觉效果和交互体验

## 🔧 实现原理

### 1. 新闻获取流程
```
分领域搜索 → 信源访问（web_fetch → agent-browser → tavily）→ 时效性验证 → 去重处理 → 分类整理
```

### 2. 数据处理流程
```
新闻数据 → 智能分类 → 热度评分 → 标签生成 → AI 摘要 → 排行榜生成 → HTML 渲染
```

### 3. 信源管理
- 严格使用信源白名单
- 中文信源：AIBase、36Kr AI、机器之心、量子位
- 国际信源：Reuters、TechCrunch、The Information、The Verge、WIRED、VentureBeat
- 官方信源：Google DeepMind Blog、OpenAI News、Anthropic Newsroom、NVIDIA Blog

## 🚀 使用方法

### 基本使用
1. 在 Trae IDE 中启用本技能
2. 输入指令：`生成今日 AI 新闻 HTML`
3. 等待技能自动完成新闻搜索、处理和渲染
4. 获取生成的 HTML 文件

### 输出文件
- 默认命名格式：`ai_news_daily_YYYYMMDD.html`
- 保存路径可自定义

## 💡 注意事项

1. **新闻时效性**：仅获取 24 小时内的新闻
2. **信源真实性**：严格使用白名单内信源
3. **内容质量**：确保新闻内容真实、完整
4. **响应式设计**：适配多种设备
